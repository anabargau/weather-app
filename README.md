# WTHR.
## What it does?
This is a simple weather app that leats you search for different locations around the world and it will show you the current weather in that specific location. You can also toggle displaying the data in Fahrenheit or Celsius and the image shown changes to represent the current weather.
## Techonolgies used:
* Javascript
* Webpack
* CSS

## How to run the app
You can simply visit the [live page](https://anabargau.github.io/weather-app/).
## What I learned
I learned how to use fetch data from APIs and make use of it and how to write asynchronous code using async/await.
